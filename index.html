About Me

Graduated in Computer Engineering at the School of Engineering of the Polytechnic of Porto (ISEP) in 2019.

I seek to improve my knowledge frequently by keeping myself updated, reading the latest articles in the field, and participating actively in various online training.

I consider myself a professional, with the responsibility of representing an added value for the employer and for society, through the application of best practices in the use of IT technologies.

My expectations are to achieve an appropriate position to my qualifications, in the field of software and web development.

My Projects

In this section I'm sharing some of the projects I have been working on throughout my training and career.

End Date: 01/2022

Meeting room scheduling portal developed in PHP for CRPG

Web-portal developed in PHP for CRPG, for meeting room scheduling management, with access roles and authentication. The portal allows to add, edit, remove rooms, and manage the bookings. Skills: PHP, SQL, Javascript, HTML, CSS, Bootstrap
End Date: 01/2021

Performance review system developed in PHP for CRPG

Participation in the implementation of an application to manage annual performance evaluation of employees for CRPG. The web-based application was developed with PHP and allows employees to set their individual goals and bosses to evaluate performance for each of the goals. Skills: PHP, SQL, Javascript, HTML, CSS
End Date: 07/2019

Institutional web template for CRPG

Creation of an institutional web template to serve as a standard for internal development projects. Skills: HTML, CSS, Bootstrap, Javascript
End Date: 05/2019

React Burger Project

Project developed in React.js following an online training, consisting of a platform with user authentication, for customizing a burger, buying and managing purchase orders. Skills: JavaScript, ES6, Spread, React, Redux, Firebase, Hooks, React Router
End Date: 01/2019

Final project of the degree in computer engineering in ISEP

Development of restful web application in scope of final project of the degree in computer engineering deployed at AWS for the registration of domestic animals, enabling the owners to register their animals and centralized management services. It was used an API in Nodejs for the backend, Angular for the frontend, database MongoDB, architecture in layers and MVC pattern. Skills: Nodejs, Angular, MongoDB, MVC, JS, AWS, HTML, CSS, GIT
End Date: 02/2018

Implementation of asset management opensource platform GLPI

Beta implementation of asset management opensource platform GLPI to study the possibility of adopting a free platform capable of responding to the needs of the team and the clients. Custom configuration of the UI and test of ticket registration, notifications, user import via LDAP, inventory management with Fusion Inventory, plugins, and template development. Skills: HTML, CSS, PHP, GLPI
